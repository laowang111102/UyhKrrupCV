# 基于Django的图书推荐管理系统 python302

## 项目概述

本项目是基于Python的Django框架开发的一款图书推荐管理系统。系统通过用户评分协同过滤算法为普通用户推荐图书，同时提供了用户登录注册、图书评分点赞以及管理员对数据的增删改查等功能。

## 技术栈

- 后端框架：Django
- 数据库：MySQL
- 开发工具：PyCharm等Python开发工具
- 前端样式：Bootstrap

## 功能模块

### 普通用户

- 登录注册
- 系统推荐图书
- 基于用户评分协同过滤算法推荐
- 对书籍点赞评分

### 管理员

- 对所有数据增删改查

## 系统角色

- 普通用户：使用系统的基本功能
- 管理员：负责系统数据的维护

## 运行环境

- Python环境
- MySQL数据库

## 部署步骤

1. 安装Python环境
2. 安装MySQL数据库
3. 导入SQL文件至MySQL数据库
4. 使用PyCharm等工具运行Django项目
